# Weather Application

A fully functional Weather web application where we can see the temperature & weather of different cities,
Open-weather-map's API was used to fetch and show the data

---

## Live Demo
- **https://mern-weather-application-1.onrender.com/** 

---
## Tech Stack
 
| Layer | Technology |
|-------|-----------|
| Frontend | React.js + Vite + Tailwind CSS |
| Backend | Node.js + Express.js |
| HTTP Client | Axios |
| Deployment | Render |

---

## Backend Setup
```bash
cd backend
npm install env cors axios express                                                        #install required packages
node server.js                                                                            #to run the backend
```

## Frontend Setup
```bash
npm create vite@latest frontend                                                            #to create vite react app
cd frontend
npm run dev                                                                                #to run the frontend
```
